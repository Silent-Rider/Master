Язык программирования: C++ (OpenMP)
Исполняемый файл: openmp_app.exe

ТРЕБОВАНИЯ К ОКРУЖЕНИЮ
Для запуска программы требуется только наличие исполняемого файла openmp_app.exe.

ИНСТРУКЦИЯ ПО ЗАПУСКУ
Запуск выполняется из командной строки с передачей аргументов.

Синтаксис:
openmp_app.exe <N> <K> [numThreads]

Где аргументы:
<N> — целевое число для разбиения (целое число > 0).
<K> — количество слагаемых (целое число > 0).
[numThreads] — (необязательно) количество потоков OpenMP.
Если параметр не указан, используется максимальное число доступных потоков.

ПРИМЕРЫ ЗАПУСКА

Автоматическое определение числа потоков:
openmp_app.exe 160 13

Запуск с фиксированным числом потоков (4):
openmp_app.exe 180 14 4

ВЫХОДНЫЕ ДАННЫЕ
Программа выводит в консоль:
- параметры запуска (N, K, количество потоков)
- результаты последовательного и параллельного вычислений
- время выполнения
- результат проверки корректности
- коэффициент ускорения

ОБРАБОТКА ОШИБОК
При передаче некорректных аргументов программа выводит сообщение об ошибке и завершает работу.
