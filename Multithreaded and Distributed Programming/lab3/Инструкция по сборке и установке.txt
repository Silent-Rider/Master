Язык программирования: C++ (CUDA)
Исполняемый файл: cuda_app.exe

ТРЕБОВАНИЯ К ОКРУЖЕНИЮ
Для запуска программы необходимо:
1. Наличие видеокарты NVIDIA с поддержкой технологии CUDA.
2. Установленные драйверы видеокарты.
3. Установленный CUDA Toolkit (для корректной работы рантайма).
4. Исполняемый файл cuda_app.exe, скомпилированный командой:
nvcc -o cuda_app.exe cuda_app.cu -arch=sm_75
(Примечание: параметр -arch=sm_75 соответствует архитектуре вашей видеокарты и может отличаться для других устройств).

ИНСТРУКЦИЯ ПО ЗАПУСКУ
Запуск выполняется из командной строки с передачей аргументов.
Синтаксис:
cuda_app.exe <N> <K> [blockSize]
Где аргументы:
<N> — целевое число для разбиения (целое число > 0).
<K> — количество слагаемых (целое число > 0).
[blockSize] — (необязательно) количество потоков в блоке CUDA.

Если параметр не указан, используется значение по умолчанию 256.
Допустимый диапазон: от 1 до 1024. Рекомендуется использовать значения, кратные 32 (32, 64, 128, 256, 512).

ПРИМЕРЫ ЗАПУСКА
Запуск с параметром по умолчанию (blockSize=256):
cuda_app.exe 2500 4
Запуск с явно указанным размером блока (32 потока):
cuda_app.exe 5000 4 32
Запуск малой тестовой задачи:
cuda_app.exe 500 3 128

ВЫХОДНЫЕ ДАННЫЕ
Программа выводит в консоль:
1. параметры запуска (N, K, конфигурация сетки и блоков CUDA)
2. результаты последовательного (CPU) и параллельного (GPU) вычислений
3. время выполнения для каждого варианта
4. коэффициент ускорения (Speedup)

ОБРАБОТКА ОШИБОК
При передаче некорректных аргументов (отсутствие параметров, нечисловые значения, значения <= 0, blockSize > 1024) программа выводит сообщение об ошибке с инструкцией по правильному использованию и завершает работу.
При ошибках оборудования или выполнения CUDA-функций программа выводит детальное описание ошибки (код, файл, строка) и завершает работу.
